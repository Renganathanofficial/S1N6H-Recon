2. Subfinder (https://github.com/projectdiscovery/subfinder)
3. Assetfinder (https://github.com/tomnomnom/assetfinder)
4. Amass (https://github.com/OWASP/Amass)
5. Sublist3r (https://github.com/aboul3la/Sublist3r)
